Help

You can get help with CsoundQt on the main Csound Mailing List, but for CsoundQt specific issues, you can join the CsoundQt Users Mailing List, for ideas, bug reports and suggestions. This mailing list is also available as a Forum on Nabble.

Documentation

Currently there is only a Quick Reference Guide for CsoundQt, but hopefully CsoundQt is intuitive enough that it should be enough to get you started. This guide is also available on the Help menu in the CsoundQt application.

However there are two excellent chapters in the Csound FLOSS manual:

  • http://en.flossmanuals.net/csound/qutecsound/ describes in detail the configuration menu (of version 0.7.3)
  • http://en.flossmanuals.net/csound/c-python-in-csoundqt describes how to use Python in CsoundQt.

Wiki

The CsoundQt Wiki is open for contributions. Have a look at the old CsoundQt Wiki, but please avoid adding information there as it might be removed in the future.
